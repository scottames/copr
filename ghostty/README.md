## Installation

1. Enable copr repo

```bash
sudo dnf copr enable scottames/ghostty
```

  - Substitute `dnf` for `yum` if desired

2. (Optional) Update package list

```bash
sudo dnf check-update
```

3. Install

```bash
sudo dnf install ghostty
```
