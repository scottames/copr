## Installation

1. Enable copr repo

```bash
sudo dnf copr enable scottames/swww
```

- Substitute `dnf` for `yum` if desired

1. (Optional) Update package list

```bash
sudo dnf check-update
```

1. Install

```bash
sudo dnf install swww
```
